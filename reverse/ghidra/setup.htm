Ghidra

安裝環境

Ref. https://ghidra-sre.org/
Ref. https://www.cjavapy.com/article/90/

開源的Reversing Tool，聽說是NSA之前在使用的，司徒測試一下，發現具有直接反譯成C語言的功能，即使是MIPS執行檔案也是，而相較於付費的IDA Pro，Ghidra真的是相當佛心！因此，司徒決定先介紹一下安裝過程，當作日後使用筆記。

1. Debian x64
2. 安裝OpenJDK 11 

$ cd
$ wget https://download.java.net/java/GA/jdk11/13/GPL/openjdk-11.0.1_linux-x64_bin.tar.gz
$ sudo mkdir /usr/local/jdk
$ tar -zxf openjdk-11.0.1_linux-x64_bin.tar.gz -C /usr/local/jdk
$ echo "JAVA_HOME=/usr/local/jdk/jdk-11.0.1" | sudo tee -a ~/.bash_profile &&echo "export JAVA_HOME" | sudo tee -a ~/.bash_profile
$ echo "PATH=$PATH:$JAVA_HOME/bin" | sudo tee -a ~/.bash_profile &&echo "export PATH" | sudo tee -a ~/.bash_profile &&source ~/.bash_profile

3. 安裝Ghidra 

$ cd
$ wget https://ghidra-sre.org/ghidra_9.0.4_PUBLIC_20190516.zip
$ unzip ghidra_9.0.4_PUBLIC_20190516.zip
$ cd ghidra_9.0.4
$ ./ghidraRun
